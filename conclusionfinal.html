Conclusión Final - Saberes 2024

Mi experiencia en la materia "Organización de Computadoras" ha sido muy enriquecedora y me ha permitido obtener una visión amplia y sólida de cómo funciona una computadora desde su base, tanto a nivel de hardware como de software. Aquí te doy una opinión de lo que aprendí en cada parte de la materia:

Primer Cuatrimestre:

1. Hardware y Software

Aprender sobre las diferencias y relaciones entre hardware y software fue fundamental, ya que me permitió comprender cómo los componentes físicos de la computadora interactúan con los programas que permiten su funcionamiento. La distinción entre ambos y cómo se complementan me ayudó a entender la arquitectura global de un sistema computacional.

2. Organización y Arquitectura de Computadora

Este tema fue clave para entender cómo se estructuran los sistemas de cómputo. La arquitectura de Von Neumann, por ejemplo, me mostró cómo se organiza la memoria, el procesador y los buses para asegurar que los datos y las instrucciones se procesen de manera eficiente. Es fascinante cómo todo en una computadora se organiza para hacer posibles tareas complejas con tan solo unos pocos componentes.

3. Dispositivos de Entrada, Salida y Mixtos

Estudiar los dispositivos de entrada, salida y mixtos fue esencial para entender cómo los usuarios interactúan con la computadora. Me dio una perspectiva más clara de cómo el hardware se conecta con el software para procesar la información y entregarla de vuelta al usuario de manera comprensible.

4. Arquitectura de Von Neumann

La arquitectura de Von Neumann me brindó una comprensión más profunda de cómo las computadoras gestionan la información y las instrucciones. Es interesante cómo este modelo sigue siendo relevante hoy en día, y me permitió visualizar de manera más clara la relación entre la CPU, la memoria y los buses.

5. Organización y Buses de Datos

Aprender sobre los buses de datos fue importante para entender cómo se transmite la información dentro de la computadora. Es algo que muchas veces damos por sentado, pero sin los buses de datos, el procesamiento y la comunicación entre componentes no sería posible.

Segundo Cuatrimestre:

1. Código Binario

El conocimiento del sistema binario fue clave para entender cómo las computadoras representan la información a nivel de bits. Fue un concepto esencial para adentrarme en la comprensión de cómo se almacenan y procesan los datos.

2. Pasaje de Unidades

Aprender a convertir entre sistemas numéricos (decimal a binario y viceversa) me ayudó a visualizar cómo las computadoras manejan números y cómo se almacenan los valores en la memoria. Es algo básico, pero fundamental para trabajar con cualquier tipo de programación o manejo de datos a bajo nivel.

3. Uso de App Inventor para Crear una Calculadora Binaria

Esta parte del curso fue muy divertida y educativa. Usar App Inventor para crear una calculadora que realizara conversiones de números binarios y decimales me permitió aplicar lo aprendido de manera práctica. Esta experiencia también me dio una introducción al desarrollo de aplicaciones móviles, algo que no había explorado antes.

4. HTML y su Relación con la Organización de la Computadora

A través del estudio de HTML, entendí cómo la información y el contenido de las páginas web son organizados y procesados por las computadoras. Aunque HTML no tiene una relación directa con el hardware, sí está estrechamente relacionado con cómo los sistemas operativos y los navegadores interpretan y presentan la información al usuario.

5. ¿Qué es HTML?

El aprendizaje de HTML me permitió entender la estructura básica de las páginas web y cómo se organizan los datos dentro de ellas. Fue un paso esencial en el aprendizaje del desarrollo web.

6. ¿Qué es CSS?

Aprender CSS fue un complemento perfecto para HTML, ya que me permitió comprender cómo se estilizan las páginas web y cómo separar la estructura (HTML) de la apariencia (CSS). Esto me abrió la puerta a la creación de páginas web más dinámicas y visualmente atractivas.

7. Uso de Visual Studio Code

Visual Studio Code fue la herramienta que utilicé para escribir y editar mis proyectos de páginas web. Aprender a usar este editor me dio una gran ventaja en el desarrollo web, ya que me permitió ser más eficiente, utilizar extensiones y tener un mejor control sobre el código.

Conclusión:

En general, la materia me proporcionó una comprensión profunda sobre cómo se organiza una computadora y cómo los datos son procesados y presentados al usuario. A través de la práctica con herramientas como App Inventor, HTML, CSS y Visual Studio Code, pude aplicar lo aprendido de manera práctica, lo que me permitió no solo adquirir conocimientos teóricos, sino también desarrollar habilidades que puedo usar en proyectos reales. Creo que lo más importante que aprendí es cómo todo en una computadora está interconectado y cómo la programación y el hardware deben trabajar en conjunto para hacer funcionar un sistema.
