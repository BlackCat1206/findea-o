1. Código Binario

El código binario es un sistema de representación numérica que utiliza solo dos símbolos: 0 y 1. Estos números son conocidos como "bits" (dígitos binarios) y son la base de la información procesada por las computadoras. Cada bit puede estar en uno de dos estados: 0 (apagado) o 1 (encendido).

Los sistemas binarios son fundamentales para las computadoras porque los circuitos electrónicos que las componen funcionan a partir de señales eléctricas, que se pueden representar fácilmente con dos estados: "encendido" (1) y "apagado" (0).

2. Pasaje de Unidades (de Decimal a Binario y de Binario a Decimal)

De Decimal a Binario

Para convertir un número decimal a binario, se realiza un proceso de división sucesiva entre 2. En cada paso, se toma el residuo de la división y se anota. El número binario resultante es la secuencia de restos leída de abajo hacia arriba.

De Binario a Decimal

Para convertir un número binario a decimal, se multiplica cada bit por su correspondiente potencia de 2, comenzando desde la derecha (la posición más baja, que es 2⁰). Luego, sumamos los resultados.

3. App Inventor: ¿Qué es y para qué lo utilicé?

App Inventor es una plataforma de desarrollo de aplicaciones móviles creada por Google y ahora gestionada por el Instituto de Tecnología de Massachusetts (MIT). Permite a los usuarios, incluso sin experiencia en programación, crear aplicaciones móviles de manera visual, usando un sistema de bloques en lugar de escribir código tradicional.

¿Para qué lo utilicé? Como estudiante, utilicé App Inventor para crear una calculadora que permitiera convertir números de decimal a binario y de binario a decimal. Para hacer esto, configuré bloques que tomaran la entrada del usuario (números en decimal o binarios), los procesaran y luego devolviera el resultado adecuado. App Inventor fue útil porque me permitió desarrollar la aplicación de manera sencilla, sin necesidad de aprender complejos lenguajes de programación, y enfocarme en la lógica del problema.

4. ¿Qué tiene que ver HTML con la organización de una computadora?

El HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar contenido en la web, como páginas web. En cuanto a la organización de la computadora, HTML no tiene una relación directa con el hardware, pero tiene que ver con cómo los datos se organizan y se muestran en una computadora, especialmente en el contexto de la web. HTML permite estructurar la información que luego es procesada por el navegador web, que es una aplicación que se ejecuta en el hardware de la computadora.

En resumen, HTML organiza cómo se presenta la información en la pantalla, mientras que la organización de la computadora se refiere a cómo el hardware y el software gestionan y procesan esa información.

5. ¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Se compone de una serie de etiquetas que indican al navegador cómo debe organizar y mostrar el contenido. Por ejemplo, el contenido puede estar estructurado en párrafos, encabezados, listas, imágenes, enlaces, etc.

6. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para definir la apariencia de los elementos HTML en una página web. Mientras que HTML se encarga de la estructura y contenido de la página, CSS se encarga de su diseño, como colores, fuentes, márgenes, alineación, etc.

7. Visual Studio Code: ¿Qué es y para qué se utiliza?

Visual Studio Code (VS Code) es un editor de código fuente desarrollado por Microsoft. Es una herramienta muy popular entre los desarrolladores por su flexibilidad, rendimiento rápido, y la amplia gama de extensiones que permite integrar para soportar distintos lenguajes de programación y tecnologías.

¿Para qué se utiliza? Como estudiante, utilicé Visual Studio Code para crear páginas web y escribir código en lenguajes como HTML, CSS, JavaScript y otros. Esta herramienta es ideal para desarrollar proyectos web porque tiene características como:

  • Resaltado de sintaxis para facilitar la escritura de código.
  • Extensiones que permiten añadir funcionalidades como autocompletado, depuración y control de versiones.
  • Integración con Git para gestionar el código fuente y colaborar en proyectos.
